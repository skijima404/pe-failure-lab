## Failure Mode Statement
The roadmap for continued acceleration is not spoken early enough, so the first win lands without a shared expectation of what it should lead to next.

## Context
This tends to emerge when the effort focuses on getting to the first visible success, but does not communicate the broader continuation path before that success arrives.

## Mechanism
- People hear about the first win, but not about the sequence that should follow it.
- The effort reaches an early success without having already framed it as the start of a longer arc.
- As a result, momentum drops because participants experience the win as a stopping point instead of as a transition point.

## Countermove
- Speak about the post-win path before the first win lands.
- Frame the first win explicitly as proof that unlocks the next phase, not as the finish line.
- Treat roadmap silence around the first win as a momentum risk.
