# Cross-Cutting Principles

This directory stores principles that apply across multiple Kotter-aligned success criteria.

These principles are not themselves steps in the change model. They act as reusable interpretive lenses that help explain why similar failures recur across multiple stages.

## Intended Use
- Use these nodes to capture durable concerns that appear across many `success_criteria`, `symptom`, and `failure_mode` assets
- Prefer putting the principle here once instead of repeating the same full rationale inside many step-local nodes
- Keep step-local nodes focused on how the principle breaks in a specific stage

## Frontmatter Convention
- `id`: stable identifier such as `xp-001`
- `type`: `cross_cutting_principle`
- `applies_to_success_criteria`: list of `sc-###` ids
- `illustrated_by`: list of representative `rf-###` or `fm-###` ids
